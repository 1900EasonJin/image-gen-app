# 图片生成应用 — 右侧已生成图片面板改进计划

## 一、目标

在画布右侧新增一个**已生成图片缩略图面板**，替代现有的 44px 窄工具栏。用户每次生成图片后，缩略图自动追加到此面板中，方便浏览历史、快速回看、选中放大。

---

## 四、功能要点

| 功能 | 说明 |
|------|------|
| 自动生成缩略图 | 每次 `imagesGenerated` 事件触发后，新图片自动追加到面板底部 |
| 点击预览 | 点击缩略图打开 lightbox 全屏预览（复用现有 lightbox 组件） |
| 选中高亮 | 最近生成的图片自动带 `.active` 边框高亮 |
| 添加参考图 | 面板顶部 `+` 按钮，复用现有上传参考图逻辑 |
| 滚动浏览 | 图片过多时面板内可滚动 |
| 空状态 | 无图片时面板显示灰色提示「暂无生成记录」 |
