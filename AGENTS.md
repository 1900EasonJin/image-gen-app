# 1 AGENTS.md — 1900th 的日常 Agent 配置文件

> 将此文件放在项目工作目录（如 `~/Downloads/同步空间/Pi Agent/AGENTS.md`），
> Pi 启动时自动加载。每次对话 Agent 都会读取这些偏好，无需重复说明。

---

## 1.1 语言与交流

- 始终使用中文请求与回复
- 回复简洁直接，不要啰嗦，不要过度解释
- 目标不确定时主动提问明确，不要猜测

## 1.2 工作习惯

- 输出文件放在当前工作目录，不要到处散落，使用文件夹归类（项目中的文件如果较多就需要提供前缀。如：1.，2.，3.）
- 创建 Markdown 文件时结构清晰、有目录
- 重要产出同时给出可预览版本（HTML 打开即看）
- 做完事情主动提醒我检查效果

## 1.3 常用技能

- 做 PPT 时优先使用 `/skill:ppt-master`
- 操作 Word 用 `/skill:docx`，Excel 用 `/skill:xlsx`
- 识别图片用 `/skill:deepseek-vision`
- 上网搜索用 `/skill:tavily-search`
- 复杂需求先讨论再动手：`/skill:brainstorming`

## 1.4 MiMo Code 配置

- MiMo Code 的详细配置请查看 `MIMO_CODE.md`
- MiMo Code 没有 Pi Agent 的技能系统，但有更强大的代码能力和记忆系统
- 使用 MiMo Code 时，参考 MIMO_CODE.md 中的能力对照表

## 1.5 记住

- 我主业不是写代码，Agent 是为了辅助日常工作效率
- 需要写代码时（vibe coding）按标准流程走
- 结果导向：说做了什么、放在哪了，不要大段绕弯子
